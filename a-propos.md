# En Francais

## 1- gestion d' intervention

### Existant
* GPI : un systeme web informatique pour l' admin (informatique)


### Scenario
- Si un personnel rencontre un probleme:
    * Soit il appelle par telephone la responsable
        - inserer manuellement en ticket
    * Soit il envoie un email
        - enregistrer automatiquement dans le systeme en ticket

- Le responsable update le statut si besoin (par exemple en cours, planifie ,... )

- Le responsable va faire l' intervention 

- C' est quand il revient qu il rentre dans le systeme et apres update le statut et dire les actions qu il a fait 
(ces actions est parfois oublier par le responsable ; l' intervention peut etre dans entreprises du membre du groupe donc le responsable se deplace pour faire l' intervention)


### Application mobile (ce qu' on veut faire)
- Le responsable (celui qui va faire l' intervention ) recoit une notification quand il y a une nouvelle tiket (intervention)

- Le responsable update le statut si besoin (par exemple en cours, planifie ,... )

- Le responsable va faire l' intervention 

- Il fait un rapport de ce qu il a fait

- Il demande au personnel concerner de valider son rapport par un signature

- Les informations dans le systeme sera updater automatiquement et le tciket sera clos
